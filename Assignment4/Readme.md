# Assignment 4: IPFS Integration

## Objective

To upload and retrieve files using IPFS.

## Tools Used

* Pinata (IPFS pinning service)
* Node.js

## Process

### Upload File

* Used Pinata API to upload file
* Received CID (Content Identifier)

### Retrieve File

* Accessed file using IPFS gateway:
  https://gateway.pinata.cloud/ipfs/CID
